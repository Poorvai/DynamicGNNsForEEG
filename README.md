# DynamicGNNsForEEG
This is my work under Dr. Deepak Mishra at IIST, Thiruvananthapuram. It was a research project, and I was tasked with exploring various ways to improve the classification models for Seizure prediction. 

## Introduction:
This project examines the effectiveness of Dynamic Graph Neural Networks (GNNs) for seizure prediction
using a publicly available EEG dataset. We evaluate different node feature encoding strategies, graph
construction methods, and dynamic modeling approaches
